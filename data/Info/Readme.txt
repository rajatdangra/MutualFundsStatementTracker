Instructions for server side automation - selenium-server-standalone.3.9.1.1 jar package is used.

Enable copy to output for selenium-server-standalone and chromedriver exe. They both should be on the same directory (bin).

Start server by CMD/Powershell in bin folder (published) - execute command: java -jar selenium-server-standalone-{VERSION}.jar

Start web application - you will see the automation.
